## Typeface

IBM Plex™ is our new corporate typeface. It replaces all usage of previous type faces, including Helvetica Neue, Lubalin, Badoni, Janson, and in some web cases, Arial and Georgia.

There are currently 8 weights wit h italics in Sans, Serif and Mono versions. When representing code or data, the Mono version is recommended. Mono can also be used outside of the coded world to create a unique look.
